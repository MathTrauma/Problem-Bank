\numbering 삼각형 $ABC$에서 변 $BC$의 점 $B$ 쪽으로의 연장선 위에 점 $D$를 $AB = BD$가 되도록 잡자. 각 $C$의 외각의 이등분선이 직선 $AD$와 점 $K$에서 만나고, $AK = AB + BC + CA$이다. 변 $AC$의 점 $C$ 쪽으로의 연장선 위에 점 $E$를 $BC = CE$가 되도록 잡자. 직선 $AD$와 삼각형 $ABC$의 외접원의 교점을 $L$ $(\neq A)$이라 할 때, 세 점 $B$, $E$, $L$이 일직선 위에 있다. $\angle CAB = x^\circ$ $(0 < x < 180)$라 할 때, $x$를 넘지 않는 가장 큰 정수를 구하여라.\\
