직선 $\mathrm{AB}$와 직선 $\mathrm{DC}$가 서로 평행하므로

$\angle\mathrm{BEC} = \angle\mathrm{DCE}$ (엇각)

$\angle\mathrm{DCE} = \angle\mathrm{ECB}$에서 $\angle\mathrm{BEC} = \angle\mathrm{ECB}$이므로

삼각형 $\mathrm{BCE}$는 $\mathrm{BE} = \mathrm{BC}$인 이등변삼각형이다. 즉, $\mathrm{BE} = 45$

$\mathrm{AE} = \mathrm{BE} - \mathrm{BA} = 45 - 30 = 15$

두 삼각형 $\mathrm{BHE}$, $\mathrm{DHG}$에서

$\angle\mathrm{EHB} = \angle\mathrm{GHD}$ (맞꼭지각)

직선 $\mathrm{AB}$와 직선 $\mathrm{DC}$가 서로 평행하므로

$\angle\mathrm{HBE} = \angle\mathrm{HDG}$ (엇각)

그러므로 두 삼각형 $\mathrm{BHE}$, $\mathrm{DHG}$는 서로 닮음이다.

\[
\mathrm{BH} : \mathrm{DH} = \mathrm{BE} : \mathrm{DG} = 45 : 10 = 9 : 2
\]

에서

\[
\mathrm{DH} = \dfrac{2}{11} \times \mathrm{BD}
\]

두 삼각형 $\mathrm{BHE}$, $\mathrm{DHG}$는 닮음비가 $9 : 2$이므로

\[
\triangle\mathrm{BHE} : \triangle\mathrm{DHG} = 81 : 4
\]

\[
\triangle\mathrm{BHE} = \dfrac{81}{4} \times \triangle\mathrm{DHG} = \dfrac{81}{4} \times 35
\]

두 삼각형 $\mathrm{BFE}$, $\mathrm{DFC}$에서

$\angle\mathrm{EFB} = \angle\mathrm{CFD}$ (맞꼭지각)

직선 $\mathrm{AB}$와 직선 $\mathrm{DC}$가 서로 평행하므로

$\angle\mathrm{BEF} = \angle\mathrm{DCF}$ (엇각)

그러므로 두 삼각형 $\mathrm{BFE}$, $\mathrm{DFC}$는 서로 닮음이다.

평행사변형 $\mathrm{ABCD}$에서 $\mathrm{CD} = \mathrm{AB} = 30$

\[
\mathrm{BF} : \mathrm{DF} = \mathrm{EB} : \mathrm{CD} = 45 : 30 = 3 : 2
\]

에서

\[
\mathrm{DF} = \dfrac{2}{5} \times \mathrm{BD}, \quad \mathrm{BF} = \dfrac{3}{5} \times \mathrm{BD}
\]

\[
\mathrm{FH} = \mathrm{DF} - \mathrm{DH} = \dfrac{2}{5} \times \mathrm{BD} - \dfrac{2}{11} \times \mathrm{BD} = \dfrac{12}{55} \times \mathrm{BD}
\]

두 삼각형 $\mathrm{EBF}$, $\mathrm{EFH}$에서 밑변을 각각

선분 $\mathrm{BF}$, 선분 $\mathrm{FH}$라 하면 두 삼각형의 높이는 같다.

$\mathrm{BF} = \dfrac{3}{5} \times \mathrm{BD}$, $\mathrm{FH} = \dfrac{12}{55} \times \mathrm{BD}$이므로

\[
\mathrm{BF} : \mathrm{FH} = \dfrac{3}{5} : \dfrac{12}{55} = 33 : 12 = 11 : 4
\]

$\triangle\mathrm{BHE} = \dfrac{81}{4} \times 35$에서

\[
\triangle\mathrm{EBF} : \triangle\mathrm{EFH} = 11 : 4
\]

따라서

\[
\triangle\mathrm{EFH} = \dfrac{4}{15} \times \triangle\mathrm{EBH} = \dfrac{4}{15} \times \dfrac{81}{4} \times 35 = \dfrac{81 \times 35}{15} = 189
\]
