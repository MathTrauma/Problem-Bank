선분 $\mathrm{AD}$가 각 $\mathrm{BAC}$를 이등분하므로

\[
\angle\mathrm{BAD} = \angle\mathrm{DAC}
\]

각 $\mathrm{CBA}$와 각 $\mathrm{CDA}$는 호 $\mathrm{CA}$에 대한 원주각이므로

\[
\angle\mathrm{CBA} = \angle\mathrm{CDA}
\]

그러므로 두 삼각형 $\mathrm{ABE}$, $\mathrm{ADC}$는 서로 닮음이다.

선분 $\mathrm{DE}$의 길이를 $x$라 하면

선분 $\mathrm{AE}$의 길이는 $6-x$

\[
\mathrm{AB} : \mathrm{AD} = \mathrm{AE} : \mathrm{AC}
\]

에서

\[
3 : 6 = (6-x) : 5
\]

\[
3 \times 5 = 6(6-x)
\]

\[
15 = 36 - 6x, \quad 6x = 21
\]

\[
x = \dfrac{21}{6} = \dfrac{7}{2}
\]

따라서 $x = \dfrac{7}{2}$
