\begin{problem}
삼각형 $ABC$에서 $\angle B = 45^\circ$, $\angle C = 60^\circ$, $BC = 20$이다. 
변 $BC$를 지름으로 하는 원이 변 $AB, AC$와 만나는 점을 각각 $D(\neq B)$, $E(\neq C)$라 하고, 
선분 $BD, BE, CE, CD$의 중점을 각각 $P, Q, R, S$라 하자. 
사각형 $PQRS$의 넓이를 $a + b\sqrt{3}$이라 할 때, $a + b$의 값을 구하여라. (단, $a, b$는 유리수)
\end{problem}
