\numbering  $\angle B = 90^\circ$ 인 삼각형 $ABC$ 는 $\overline{AB}=4, \overline{BC}=3$ 을 만족한다.
원 $O_1$ 은 $AB \mybreak AC$ 와 접하는 원이고, 원 $O_2$ 는 $BC \mybreak AC$ 와 접하는 원이다.
또, 원 $O_1$ 과 $O_2$ 는 반지름의 길이가 같고 서로 외접한다.
원 $O_1$ 의 반지름의 길이를 $\frac n m$ 이라 할 때, $m+n$ 을 구하여라. 
(단, $m,n$ 은 서로소인 양의 정수)
