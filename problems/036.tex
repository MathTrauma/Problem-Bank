\numbering 각 $B$ 가 예각인 삼각형 $ABC$ 에서 $\overline{BC}=8, \overline{AC} = 3 \overline{AB}$ 이다.
삼각형 $ABC$ 의 외접원과 점 $A$ 에서 접하는 직선이 선분 $BC$ 의 수직이등분선과 점 $D$ 에서 만나고 $\overline{AD}=6$ 이다.
삼각형 $BCD$ 의 외접원의 반지름의 길이를 $r$ 이라 할 때, 
$7r^2$ 의 값을 구하여라.\\
